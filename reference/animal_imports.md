# Example Animal Import Data

A dataset containing information about animal imports from three
different regions

## Usage

``` r
animal_imports
```

## Format

### animal_imports

A data frame with 3 rows and 4 columns:

- origin:

  Region of origin (nord, south, east)

- farms_n:

  Number of farms exporting animals

- animals_n_mean:

  Mean number of animals exported per farm

- animals_n_sd:

  Standard deviation of animals exported per farm

## Source

Simulated data for demonstration purposes
